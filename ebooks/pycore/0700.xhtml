Note that the examples in this chapter require a Windows operating system; they will not work on Apple computers running Microsoft Office for Mac.


This chapter represents a departure from most other sections of this book, meaning that instead of focusing on developing networked, GUI, Web, or command-line-based applications, we’ll be using Python for something completely different: controlling proprietary software, specifically Microsoft Office applications, via Component Object Model (COM) client programming.
