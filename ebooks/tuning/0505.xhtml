Web Resources for /proc

URLDescription
http://www.kernel.orgKernel source. Several files in the /Documentation directory of the kernel source provide more information about each /proc entry. The proc.txt file in the /filesystems directory is one such example of a file that contains additional information about the /proc file system.
http://perso.wanadoo.fr/sebastien.godard/sysstat utilities (iostat, sar, sadf, mpstat, and sa). The sysstat utilities are a collection of performance-monitoring tools for Linux. These tools get their system information from the /proc file system.
