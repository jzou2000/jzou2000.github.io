Finding Kprobes That Are Active on the System

First the kernel must have Magic SysRq Key support configured (see Figure 12.3 for the menu where this option is configured for the kernel).

The echo command can be used to enable SysRq key support, as shown in Figure 12.7. To display the Kprobes active on the system, press Alt-SysRq-w. The example of /var/log/messages in Figure 12.7 shows that the sys_open Kprobe is the only active Kprobe on the system.
