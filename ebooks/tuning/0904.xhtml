Building LTT User-Level Tools

The latest revision of the user tools for TraceToolkit is version 0.9.6-pre4, and the tar file that has this version is called ltt-0.9.6-pre4.tar.bz2. Using the tar command expands the source into the ltt-0.9.6-pre4 subdirectory. We’ll use the standard ./configure, make, and make install commands to build and install the user-level tools for LTT, as shown in Figure 9.7.
