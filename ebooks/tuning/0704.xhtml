Using gdb to Display jfs_mount

Another way to disassemble the jfs_mount code is to use gdb, as shown in Figure 7.7. The jfs_mount source code is in the kernel source tree and is located in the /usr/src/linux-2.4.27/fs/jfs/jfs_mount.c file. So the first step is to start gdb with input as fs/jfs/jfs_mount.o and to use gdb’s disassemble function to display the jfs_mount routine.
