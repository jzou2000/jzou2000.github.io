Stopping Data Recording

Some of the scripts listed in the preceding section stop tracing after a specified time interval. Others allow tracing to run until the trace daemon process is stopped. You can use the following command to stop the trace daemon manually:

# pkill [tracedaemon]

The ps command can be used to find the process ID for the trace daemon. After you find it, you can use the kill command to stop the trace daemon:

# kill [tracedaemon pid]
