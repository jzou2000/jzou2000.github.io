Ways an Administrator Can Use /proc

/proc can be used for the following types of system-related tasks:

  • Performance and memory information

  • Viewing and modifying runtime parameters

  • Viewing hardware information

  • Viewing and modifying network parameters

  • Viewing statistical information

Note

You must be careful when changing parameters through the /proc interface.
