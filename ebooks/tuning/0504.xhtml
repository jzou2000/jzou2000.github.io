Summary

This chapter covered the system information available from the /proc file system and some of the key commands for performance tuning and debugging. These commands use the /proc entries to display system information. The /proc file system has a wealth of information—from information about each process to system-wide information about CPU, memory, file systems, interrupts, and partitions. Some of the /proc file system parameters can be tuned. If tuning has been done on a running kernel, make sure that each file that was tuned has the new parameter set by adding it to the distribution sysctl.conf file.
