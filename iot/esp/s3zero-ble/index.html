Full Development Cycle - Build A BLE Server/Client App

This project is used to demonstrate the full development cycle with the help of Grok.

BLE server is built on Waveshare S3 zero chip. The server exposes the embedded temperature sensor and controls on-board WS2812 RGB LED module, so no physical circuit is required, simply connect the s3 zero board to USB port.

The client is a web app that uses BLE api to get information and control the S3 chip.

Grok is used to generate the initial codes.
