ODBC Driver Manager

Enable DM/Driver log

unixODBC

odbcinst.ini

[ODBC]
Trace = yes
TraceFile = /path/to/traceFile.log

iODBC

odbc.ini

[ODBC]
Trace = 1
TraceFile = /path/to/traceFile.log

# optional
Debug = 1
DebugFile = /path/to/debugFile.log

Simba Driver

simab.${drier}.ini

[Driver]
LogLevel=6
LogPath=/path     # file name is, e.g. XM_driver.log
EnableTransactions=0

Log Level

  1. disable
  2. fatal - very severe error events that might crash
  3. errors - driver can still run
  4. warnings - potentially harmful situations
  5. info - general information
  6. debug - details for debugging
  7. trace - everything
