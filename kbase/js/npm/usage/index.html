Common Usage

Update npm itself

npm install -g npm

Update all installed packages

npm update

Install/Uninstall a package

npm install install @vue/cli -g
npm uninstall @vue/cli -g
  • -g global

List Packages

npm ls --depth=0 -g
  • –depth=0 list only toplevel package tree
  • -g global
