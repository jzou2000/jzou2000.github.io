Note

  • -regex use grep pattern by default. To use other pattern, set by -regextype, available values are
    • ed
    • emacs
    • gnu-awk
    • grep
    • posix-awk
    • awk
    • posix-basic
    • posix-egrep
    • egrep
    • posix-extended
    • posix-minimal-basic
    • sed or use an invalid value (e.g. help to get a list)
    find * -regextype posix-extended

  • get list of all source files, excluding
    • list only files (not directories)
    • skip *.[od] compile output
    • skip *.flist output of script
    • skip b/* shadow build at b/
    • skip *.tar
    # get list of all source files:
find * -type f -not -name '*.[od]' -and -not -name '*.flist' -and -not   -name '*.tar' -and -not -path 'b/*' > src.flist
tar cvf src.tar --files-from=src.flist
