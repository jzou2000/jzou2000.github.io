get glibc version

ldd --version

Alternatively, check glibc that is used by the current bash

lsof -p $$ | grep libc

Using C program

#include <stdio.h>
#include <stdlib.h>
#include <gnu/libc-version.h>

int main(int argc, char *argv[]) {
  printf("glibc version: %s\n", gnu_get_libc_version());
  return 0;
}
