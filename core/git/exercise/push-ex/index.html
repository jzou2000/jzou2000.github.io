Push Exercise

Plan

export EX=~/codex/git
export REPO=$EX/repo
mkdir -p $REPO

Setup Root Repo

$ cd $REPO
$ mkdir adam
$ cd adam
$ git init --bare

$ cd $EX
$ git clone $REPO/adam
$ # add something and make 1st commit
$ cat > readme
$ mkdir books
$ cat > books/chap1
$ git add readme books
$ git commit -m 'fist commit'

$ # push to repo
$ git push

Create A clone of Root Repo

$ cd $REPO
$ git clone adam dave --bare
