capture shortcuts action
entire screen Shift+Command+3
select area Shift+Command+4 cursor to cross, drag selection area then release
a window Shift+Command+4 press space, cursor to camera, click target window

Note:

  • files are saved on the desktop
  • to capture to clipboard, add “Control” with shortcuts in the same time, e.g. Shift+Command+4+Control.
